## English

Professional Network Diagnostics Toolkit for macOS.

### 🌟 Key Features
*   **Advanced Ping**: Real-time RTT charts, jitter analysis, distribution histogram, and audio alerts for packet loss.
*   **Traceroute**: Detailed hop-by-hop analysis with automatic geolocation and visual path sparklines.
*   **Multi-Ping**: Monitor unlimited hosts simultaneously with live stability indicators.
*   **Port Scanner**: High-speed TCP scanner with custom concurrency and port ranges.
*   **SSL/TLS Inspector**: Comprehensive certificate chain analysis, expiry tracking, and cipher verification.
*   **Network Monitoring**: Real-time bandwidth charts, Wi-Fi signal (RSSI/SNR) analysis, and live routing tables.
*   **Data Export**: Save all diagnostic results as CSV or JSON for professional reporting.

### 🚀 Technical Overview
*   **Native Swift & SwiftUI**: Built specifically for macOS 15+ using Swift 6.
*   **Zero Dependencies**: No third-party libraries, ensuring maximum security and speed.
*   **Apple Silicon Optimized**: Native performance on M1/M2/M3/M4 chips.
*   **CI/CD Ready**: Automated build and release workflows via GitHub Actions.

### 🛠 Installation
1.  Go to the [Releases](https://github.com/aderamdani/NetUtil/releases) page.
2.  Download the latest `.dmg` file.
3.  Open the DMG and drag **NetUtil** to your Applications folder.

## Bahasa Indonesia

Toolkit Diagnostik Jaringan Profesional untuk macOS.

### 🌟 Fitur Utama
*   **Advanced Ping**: Grafik latensi real-time, analisis jitter, histogram distribusi, dan peringatan suara saat paket hilang.
*   **Traceroute**: Analisis rute per hop yang detail dengan geolokasi otomatis dan grafik stabilitas visual.
*   **Multi-Ping**: Pantau host tanpa batas secara bersamaan dengan indikator stabilitas langsung.
*   **Port Scanner**: Pemindai port TCP kecepatan tinggi dengan kontrol konkurensi dan rentang port kustom.
*   **SSL/TLS Inspector**: Analisis rantai sertifikat lengkap, pelacakan masa berlaku, dan verifikasi cipher.
*   **Pemantauan Jaringan**: Grafik bandwidth real-time, analisis sinyal Wi-Fi (RSSI/SNR), dan tabel routing langsung.
*   **Ekspor Data**: Simpan semua hasil diagnostik sebagai CSV atau JSON untuk pelaporan profesional.

### 🚀 Ikhtisar Teknis
*   **Native Swift & SwiftUI**: Dibangun khusus untuk macOS 15+ menggunakan Swift 6.
*   **Tanpa Dependensi**: Tanpa pustaka pihak ketiga, menjamin keamanan dan kecepatan maksimal.
*   **Optimasi Apple Silicon**: Performa asli pada chip M1/M2/M3/M4.
*   **Siap CI/CD**: Alur kerja build dan rilis otomatis melalui GitHub Actions.

### 🛠 Instalasi
1.  Buka halaman [Releases](https://github.com/aderamdani/NetUtil/releases).
2.  Unduh file `.dmg` terbaru.
3.  Buka file DMG dan tarik **NetUtil** ke folder Applications Anda.
