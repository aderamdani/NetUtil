# Security Policy

## Supported Versions

Currently, only the latest version of NetUtil is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.3.x   | :white_check_mark: |
| < 1.3.0 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within NetUtil, please follow these steps:

1. **Do not** open a public issue.
2. Send an email to the maintainer (Ade Ramdani) or use GitHub's private vulnerability reporting if available.
3. Provide as much detail as possible, including steps to reproduce.

We will acknowledge your report within 48 hours and provide a timeline for a fix if applicable.

## Philosophy
NetUtil is designed with privacy and security in mind:
- **Zero Telemetry**: We do not collect any data.
- **Local Only**: All network diagnostics data stays on your machine.
- **Native Tools**: We use standard macOS system tools for diagnostics.
